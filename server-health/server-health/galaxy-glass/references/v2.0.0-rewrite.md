# GalaxyGlass v2.0.0 — 完全重构 (2026-05-12)

## 触发场景

用户说："你现在以你现在的样式，来重构它，用上你认为最好的布局、最好的颜色搭配、最好的代码逻辑"

## 重构范围

### 色彩体系
- 新主色: 翠绿 `#10b981` + 靛蓝 `#6366f1`（取代旧紫色 `#8b5cf6`）
- 三层背景深度: `#050d07`（最深，scrollbar/body）→ `#071109`（中间层）→ `#0a1a0f`（表面）
- 文字: `#f0fdf4` 浅绿白代替纯白，更柔和
- 状态色分层: success/warning/danger

### 布局
- 容器 max-width: 1124px → 1280px
- 卡片 min-width: 320px → 340px（大屏 380px）
- 使用 `.wrap` 容器统一水平 padding（`--container-pad: 1.75rem` 替代旧 1.5rem）

### CSS
- 30+ CSS 变量全面覆盖所有设计令牌
- 自定义深色 scrollbar
- ::selection 样式
- cardIn 入场动画 keyframes（opacity+translateY+scale，延迟递增）
- 搜索框 `:focus-within` 状态代替纯 JS `.open` 类
- 圆角变量化: `--radius-sm/md/lg/xl/full`
- 间距变量化: `--gap/gap-sm/gap-xs`

### JS 架构变化
- 从全局 script → IIFE `(function(){ 'use strict'; ... })()`
- var 代替 const/let（避免重复声明报错）
- `$()` 短函数代替重复 `document.getElementById()`
- 统一 `mergeNodeData()` 函数合并 node + recent data
- `metricRow()` 模板函数消除重复
- 图表函数统一: `getCanvasCtx()` + `drawLineChart()` + `drawNetChart()`
- 入口: `loadData()` 替代旧 `init()`，Promise 链式
- 没有 `removeEventListener` 泄漏（单次 setup）

### HTML 结构变化
- `.nodes-grid-wrap`、`.table-view-wrap`、`.region-filters-wrap` 统一 wrap 模式
- 详情页 `.detail-content-wrap` 控制 padding
- loading 态增加 spinner动画（CSS border 动画）
- 空状态 `.empty-state` 独立
- 搜索框增加 `:focus-within` 效果
- 按钮均加 `aria-label`、`role`、`tabindex`

### 已修复/改进
- 所有 transition:all → 显式声明 ✅
- 所有 width 动画 → transform:scaleX ✅
- 全局键盘事件 Enter/Space 打开详情 ✅
- SVG aria-hidden + 内联 title ✅
- 视图切换 aria-selected ✅
- 导航栏 transition 只含 background/backdrop-filter/border-color ✅
- 详情页指标卡字号从 22px 改为 15px（更协调） ✅
- 进度条高度从 6px → 5px（更细腻） ✅

```bash
# Login → upload → activate
SESS_TOKEN=$(curl -s -X POST "https://<监控面板域名>/api/login" \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"ybr52tlztwfr6b"}' | \
  python3 -c "import sys,json; d=json.load(sys.stdin); print(d['data']['set-cookie']['session_token'])")
curl -s -X PUT -H "Cookie: session_token=$SESS_TOKEN" -F "file=@/tmp/gg-v2.zip" https://<监控面板域名>/api/admin/theme/upload
curl -s -H "Cookie: session_token=$SESS_TOKEN" "https://<监控面板域名>/api/admin/theme/set?theme=GalaxyGlass"
```
