# GalaxyGlass v1.3.0 — Visible Accent Colors Throughout

> Released 2026-05-12. GitHub: `github.com/newemperor221/galaxy-glass/releases/tag/v1.3.0`

## What Changed

### 1. Navbar Title Gradient

```css
.navbar-title {
  background: var(--accent-gradient);       /* green → purple */
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
  font-weight: 700;
  text-shadow: none;
}
```

**Before**: solid white `rgba(255,255,255,0.9)` with text-shadow
**After**: gradient green→purple text. Gives the brand header a premium, polished feel.

### 2. Active Filter Chips with Gradient

```css
.chip.active {
  border-color: rgba(16,185,129,0.5);
  background: linear-gradient(135deg, rgba(16,185,129,0.15), rgba(139,92,246,0.1));
  color: var(--accent);
  box-shadow: 0 0 12px rgba(16,185,129,0.08);
}
```

**Before**: solid green-tinted background
**After**: green→purple gradient background with subtle glow

### 3. Price Badge with Gradient Background

```css
.node-footer-price {
  color: #fff;
  background: var(--accent-gradient);
}
```

**Before**: green text on transparent background, easily overlooked
**After**: white text on gradient background (green→purple), much more visible

### 4. Accent Bar — Top Decorative Line

```html
<div class="accent-bar"></div>
```

```css
.accent-bar {
  width: 80px;
  height: 3px;
  background: var(--accent-gradient);
  border-radius: 9999px;
  margin-bottom: 16px;
}
```

A thin gradient line above the stats grid. Subtle but adds brand-like polish.

### 5. Stat Card Icon Hover (green)

```css
.stat-card svg { transition: color 0.3s; }
.stat-card:hover svg { color: #10b981; }
```

**Before**: icons stayed dim gray always
**After**: icons turn green on hover, adding interactive feedback

### 6. Online Dot Pulse with Double Glow

```css
@keyframes pulse-dot {
  0%,100% { opacity:1; box-shadow:0 0 6px rgba(16,185,129,0.6),0 0 12px rgba(16,185,129,0.2); }
  50% { opacity:0.7; box-shadow:0 0 3px rgba(16,185,129,0.3),0 0 6px rgba(16,185,129,0.1); }
}
```

**Before**: single-layer glow, simple opacity pulse
**After**: double-layer glow (inner 6px + outer 12px) that also pulses in intensity

### 7. Transition Cleanup (final pass)

All remaining `transition: all` occurrences converted to explicit property lists:
- `.node-card` → `transform, box-shadow, border-color`
- `.table-row` → `transform, box-shadow, border-color`
- `.stat-card` → `border-color, box-shadow, transform`

## Design Principle from This Release

**CSS variables must be used, not just defined.** In the initial v1.2.3 release, `--accent-2` and `--accent-gradient` were added to `:root` but never applied to any visible element — the user called this a "半成品" (half-baked product). The lesson: when adding design tokens (CSS variables), apply them to at least one visible element in the same commit. Unused variables are dead code.
