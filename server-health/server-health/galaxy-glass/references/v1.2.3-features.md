# GalaxyGlass v1.2.3 — Aesthetic & Performance Features

> Released 2026-05-12. GitHub: `github.com/newemperor221/galaxy-glass/releases/tag/v1.2.3`

## 1. Accent Color System

Added a secondary accent color and gradient for visual depth:

```css
:root {
  --accent: #10b981;                    /* primary green */
  --accent-2: #8b5cf6;                  /* secondary purple */
  --accent-gradient: linear-gradient(135deg, #10b981, #8b5cf6);  /* green→purple */
}
```

**Applied to:**
- `.node-card:hover` — green border glow (`border-color: rgba(16,185,129,0.2)` + `box-shadow: 0 0 0 1px rgba(16,185,129,0.15)`)
- `.table-row:hover` — same green border glow
- `.stat-card:hover` — green border + box-shadow feedback
- `.chip.active` — already used `--accent` for border/background

**Why purple?** The green→purple gradient gives depth beyond a single green accent without being garish. The green channel (primary) is used for interactive feedback; purple (secondary) is available for decorative elements, charts, or future UI accents.

## 2. Online Status Breathing Pulse

```css
.node-status.online {
  background: #10b981;
  box-shadow: 0 0 8px #10b981;
  animation: pulse-dot 2s ease-in-out infinite;
}
@keyframes pulse-dot {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.6; }
}
```

- 2s cycle: full opacity → 0.6 opacity → full opacity
- Subtle enough not to be distracting, visible enough to distinguish alive servers at a glance
- Only affects online dots; offline dots remain solid red

## 3. Wider Card Grid for Large Screens

```css
/* Default: 320px min (was 300px) */
.nodes-grid {
  grid-template-columns: repeat(auto-fill, minmax(320px, 1fr));
}

/* 1600px+ wide screens: 360px min for more spacious cards */
@media (min-width: 1600px) {
  .nodes-grid { grid-template-columns: repeat(auto-fill, minmax(360px, 1fr)); }
}
```

**Effect**: 
- 1920px screens at 320px: 5-6 columns depending on sidebar
- 1920px at 360px: 4-5 columns, each card noticeably roomier
- The 300→320px bump alone gives ~7% more width per card even at default sizes

## 4. Stat Card Hover Feedback

```css
.stat-card {
  transition: border-color 0.2s, box-shadow 0.2s, transform 0.2s; /* was `all 0.3s` */
}
.stat-card:hover {
  border-color: rgba(16,185,129,0.2);
  box-shadow: 0 0 0 1px rgba(16,185,129,0.1);
}
```

Previously `.stat-card` had `transition: all 0.3s` but no hover effect at all — the transition was pointless overhead. Now it has a subtle green border glow on hover, consistent with card/table row hover treatment.
