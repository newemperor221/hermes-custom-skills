# GalaxyGlass v2.1.0 — 骨架屏 / 卡片精简 / OS图标CSS化

## 骨架屏加载（renderSkeletons）

在 `loadData()` 入口立即执行 `renderSkeletons()`，显示 8 张 shimmer 占位卡片。

```css
.skeleton-card {
  display: flex; flex-direction: column; gap: 12px;
  padding: 1.1rem; border-radius: var(--radius-lg);
  background: var(--bg-glass);
  backdrop-filter: blur(var(--blur-glass)) saturate(120%);
  border: 1px solid var(--glass-border);
  min-height: 160px;
}
.skeleton-line {
  height: 10px; border-radius: var(--radius-full);
  background: linear-gradient(90deg, rgba(255,255,255,0.03) 25%, rgba(255,255,255,0.07) 50%, rgba(255,255,255,0.03) 75%);
  background-size: 200% 100%;
  animation: shimmer 1.5s ease-in-out infinite;
}
@keyframes shimmer {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
```

- 骨架尺寸与实际卡片一致，避免布局偏移
- 第一行 `.skeleton-line:first-child { width: 60%; height: 14px }` 模拟标题
- 中间行 `width: 100%; height: 6px` 模拟进度条
- 最后一行 `width: 40%; margin-top: auto` 模拟 footer
- 数据到齐后 `render()` 的 `grid.innerHTML = filtered.map(renderCard)` 自动替换

## 卡片精简

v2.1.0 卡片从 5 块信息压缩到核心项：

```
之前：状态+OS图标+名称+国旗 | 硬件摘要行 | CPU/MEM/DSK 3行 | 网络1行 | uptime+流量+价格
之后：状态+OS图标+名称+国旗 | CPU/MEM/DSK 3条+NET 1条 | uptime+价格
```

去掉的：
- 硬件摘要行（OS · virt · CPU型号）→ 仅详情页保留
- 底部流量统计 → 仅详情页保留
- `.node-specs` 整个 div 删除
- 列宽：`minmax(340px, 1fr)` → `minmax(300px, 1fr)`
- gap: 10px → 8px
- padding: 1.1rem → 1rem
- metrics gap: 10px → 4px（更紧凑的进度条行）
- card-metric 每行固定高度 20px

## OS 图标 CSS ::before

完全废弃 data URI + img 标签方案。

```html
<!-- 之前：<img class="node-os-icon" src="data:image/svg+xml,..."> -->
<!-- 之后：--><span class="node-os-icon" data-os="alpine"></span>
```

```css
.node-os-icon {
  width: 16px; height: 16px;
  display: inline-flex; align-items: center; justify-content: center;
  flex-shrink: 0;
}
.node-os-icon::before { font-size: 14px; line-height: 1; }
.node-os-icon[data-os*="alpine"]::before { content: "\26F0"; color: #5e81f4; }
.node-os-icon[data-os*="debian"]::before { content: "\2666"; color: #D70A53; }
.node-os-icon[data-os*="ubuntu"]::before { content: "\262E"; color: #E95420; }
.node-os-icon[data-os*="centos"]::before,
.node-os-icon[data-os*="rhel"]::before  { content: "\25CF"; color: #9326ee; }
```

JS 中 `getOSIcon()` 替换为 `osClass(os)`，只返回 'alpine'/'debian' 等 class 名。

## 筛选按钮高亮加强

```css
.chip.active {
  border-color: var(--accent);
  background: linear-gradient(135deg, rgba(16,185,129,0.18), rgba(99,102,241,0.12));
  color: var(--accent);
  box-shadow:
    0 0 24px rgba(16,185,129,0.12),    /* 外发光 */
    inset 0 0 20px rgba(16,185,129,0.04);  /* 内发光 */
}
```

普通态维持半透明毛玻璃，选中态实色绿边框 + 渐变背景 + 呼吸发光。

## 错误态

当 `/api/nodes` 返回空时，`showError()` 渲染错误提示而非 spinner。

```js
function showError(msg) {
  var grid = $('grid-view');
  if (grid) grid.innerHTML = '<div class="error-state"><div class="error-icon">⚠️</div><span>' + msg + '</span></div>';
}
```

## 装饰条脉冲（调暗）

```css
.accent-bar {
  animation: accentPulse 8s ease-in-out infinite;
}
@keyframes accentPulse {
  0%, 100% { opacity: 0.85; }
  50% { opacity: 0.55; }
}
```

- 周期从 4s → 8s
- 变化幅度从 0.6→1.0 → 0.55→0.85
- 几乎注意不到，但移除会感觉页面死板
