## 替换要点

- 所有 Heroicons Solid SVG 用 `fill="currentColor"`（继承父元素 color），不用 `stroke`
- viewBox 统一 `0 0 24 24`
- 在 `stat-card svg` 的 CSS 中改 `color` 而不是 `stroke` 来控制颜色
- 如果要保留 `<svg>` 上已有的 inline `stroke-width` 属性，Heroicons 不需要它（fill 模式忽略 stroke）
