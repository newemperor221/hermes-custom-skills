# 色彩审计方法论 — GalaxyGlass

> 当用户问"这个颜色和背景配不配"时，执行此审计流程。

## 核心原则

**同色相 ≠ 好搭配。** accent 色如果和背景处于同一色相家族，对比度会急剧下降，视觉上"消失"。

| 色相关系 | 效果 |
|----------|------|
| ≤ 30°（同色系） | 自然融入，但不醒目 |
| 30-90°（邻近色） | 和谐但可辨 |
| 90-150°（对比色） | 视觉突出 |
| 150-180°（互补色） | 最大冲击力（适合警告） |

## 审计流程

### 1. 计算所有颜色对背景的对比度

用 Python 计算相对亮度和 WCAG 对比度：

```python
def relative_luminance(r, g, b):
    rs = r/255; gs = g/255; bs = b/255
    rs = rs/12.92 if rs <= 0.03928 else ((rs+0.055)/1.055)**2.4
    gs = gs/12.92 if gs <= 0.03928 else ((gs+0.055)/1.055)**2.4
    bs = bs/12.92 if bs <= 0.03928 else ((bs+0.055)/1.055)**2.4
    return 0.2126*rs + 0.7152*gs + 0.0722*bs

def contrast_ratio(l1, l2):
    lighter = max(l1, l2)
    darker = min(l1, l2)
    return (lighter + 0.05) / (darker + 0.05)
```

WCAG 标准：
- AAA: ≥ 7:1
- AA: ≥ 4.5:1  
- AA large: ≥ 3:1
- < 3:1 = FAIL（可读性差）

### 2. 检查色相关系

计算每个颜色 HSL 色相与背景色相的差值，判断属于哪种关系。

### 3. 重点检查项

| 检查项 | 预期 |
|--------|------|
| `--text-primary` on bg | ≥ 10:1（必须清晰） |
| `--accent` on bg | ≥ 5:1（主色要跳） |
| `--accent-2` on bg | ≥ 5:1（辅色也要可见） |
| `--text-muted` on bg | ≥ 3:1（弱级文字不能完全看不清） |
| `--accent-orange` on bg | ≥ 5:1 + 互补/对比色（警告要醒目） |
| `--danger` on bg | ≥ 4.5:1（红色要保持识别度） |

### 4. 调整策略

**accent-2 和背景同色系时**：不要只调色相，应该**调亮**（提高 lightness）。保持色相一致但亮度差拉大。例如 `#6366f1` (67% lightness) → `#818cf8` (74% lightness) 在深蓝底上对比度从 4.0:1 升到 6.3:1。

**text-muted 在深色背景上**：不要用纯黑/纯白的不透明度，应该基于背景色计算有效亮度。深底上 32% → 38% 透明度能把对比度从 2.7:1 提到约 3.5:1。

### 5. 硬编码色值追踪

当 `--accent-2` 变更时，以下位置必须同步更新（v2.5.0 经验）：

| 位置 | 之前值 | 更新值 |
|------|--------|--------|
| `:root` 变量 | `#6366f1` | `#818cf8` |
| `--accent-gradient` | `#6366f1` | `#818cf8` |
| 内存图表线色 | `#6366f1` | `#818cf8` |
| `.chip.active` 渐变右端 | `rgba(99,102,241,0.12)` | `rgba(129,140,248,0.12)` |
| `.tag-chip-alt` | `rgba(99,102,241,0.08/0.2)` | `rgba(129,140,248,0.08/0.2)` |
| `.back-to-top:hover` | `rgba(99,102,241,0.3/0.08)` | `rgba(129,140,248,0.3/0.08)` |

**教训**：accent-2 的硬编码散落在 CSS 和 JS 中，修改时必须全局搜索 `#6366f1` 和 `rgba(99,102,241` 两遍。

## 案例：v2.4.0 → v2.5.0 深空蓝黑底审计

背景色改为 `#0e152e`（深靛蓝黑，hsl 226°, 53%, 12%）后：

| 颜色 | 对比度 | 结论 |
|------|--------|------|
| `#f0fdf4` 文字主色 | 17.2:1 ✅ AAA | 薄荷白在深蓝底上略偏暖，但可读性极好 |
| `#10b981` 翠绿 accent | 7.4:1 ✅ AAA | 在蓝底上比在绿底上更亮，色差 120° |
| `#6366f1` 靛蓝 accent-2 | 4.0:1 ⚠️ AA | **同色系低对比** → 提亮到 `#818cf8` (6.3:1) |
| `#f59e0b` 琥珀警告 | 8.4:1 ✅ AAA | 蓝橙互补色 = 最大冲击 |
| `#ef4444` 红色危险 | 4.8:1 ✅ AA | 足够识别 |
| text-muted @32% | 2.7:1 ❌ FAIL | → 提到 38% (~3.5:1) |

**最终结论**：accent-2 在深蓝底上不应当用暗靛蓝（同色系消失），应当用亮紫蓝。text-muted 从 32% 提到 38%。
